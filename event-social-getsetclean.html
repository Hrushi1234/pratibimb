Get Set Clean

Get set Clean : Cleanliness is always the first priority of Pratibimb. We believe that if our environment is clean then only we can survive in this nature. Students of VJTI visit different beaches in Mumbai to clean up the garbage and unwanted waste that is spreading over the beach. All the students willingly participate in this beach clean up drive.
